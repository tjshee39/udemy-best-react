### 1. install dependencies<br>

- **frontend**

```shell
$ 01-starting-project 18> npm install
```

- **backend**

```shell
$ 01-starting-project 18\backend> npm install
```

### 2. run frontend<br>

```shell
$ 01-starting-project 18> npm run dev
```

### 3. run backend<br>

```shell
$ 01-starting-project 18\backend> npm run start
```
